# StringUtil
A browser extension to modify and manage strings

supports...

* Toggle Case
* UPPER CASE
* lower case
* Leading Caps
* camelCase
* PascalCase
* kebab-case
* snake_case
* Delimiter
* De-Whitespaced
* w h i t e s p a c e d
* Reverse
* Random Case
